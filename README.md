# console.image.js
### To use: ```console.image(url, [optional params])```
  
#### Required parameters:
* **Url**  
#### Optional parameters:
* **Size:** (default - 0.3)
* **Returning text:** (default - undefined)
* **Covering:** (contain/cover/auto, default - contain)
